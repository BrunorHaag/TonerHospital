
# Deploy Firebase (Execute em ordem)

## 1. Instalar CLI
```bash
curl -sL https://firebase.tools | bash
firebase login
```

## 2. Deploy Regras DB
```bash
firebase deploy --only database --project toners-ahsj
```

## 3. Deploy Hosting
```bash
firebase deploy --only hosting --project toners-ahsj
```
